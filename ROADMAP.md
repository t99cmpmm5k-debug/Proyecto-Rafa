# Corredor Sólido — Sprint 1

## Base estable

V51 Mapa Premium. El apartado Plan queda congelado y no se modifica.

## Fase 1 — OCR de capturas

Objetivo de esta entrega:

- Seleccionar una o varias capturas desde Running.
- Leer el texto con Tesseract OCR.
- Detectar de forma preliminar Garmin, Amazfit o Apple Watch.
- Mostrar el texto reconocido y la confianza de cada captura.
- No interpretar métricas todavía.
- No guardar ni modificar entrenamientos.

## Siguiente fase

Crear el parser Garmin para extraer distancia, tiempo, ritmo, frecuencia cardiaca, cadencia, calorías y temperatura.
